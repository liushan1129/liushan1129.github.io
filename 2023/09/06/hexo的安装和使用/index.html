hexo的安装和使用

John Doe
2023-09-06

前言

文档编辑使用markdown语言，语法格式学习网站: https://m.runoob.com/markdown

安装

hexo 是基于nodejs开发的网站框架，需要先安装node.js

注意：windows cmd下敲命令可能会提示没权限，此时可以使用管理员身份运行cmd即可

nodejs

①node官网：http://nodejs.cn/

cnpm安装

npm install -g cnpm –registry=https://registry.npm.taobao.org

git安装

git官网：https://git-scm.com/downloads/

全局安装脚手架hexo-cli

npm install -g hexo-cli

利用hexo创建项目

默认安装在所在文件夹
hexo init myhexo

安装依赖

进入到myhexo目录下
npm install 或者 yarn (如果提示没权限可使用cnpm试试)

启动项目

hexo s (或 hexo server)

使用

新建一篇博客

hexo n 第一篇博客
在myhexo目录下生成“/source/_posts/第一篇博客.md”,使用或者文档编辑器打开，编辑博客内容后保存,退回到myhexo目录。

执行清理

hexo clean

执行生成

hexo g

执行启动

hexo s

推送至github

在github创建一个仓库,注意仓库名[必须与github的用户名一致 后面加上.github.io]即为后续访问博客的地址

安装git插件

cnpm install –save hexo-deployer-git

修改 _config.yml文件

Deployment
deploy:
type: git
repo: https://github.com/xxxx/xxxx.git
branch: master
*

部署到remote

hexo d (hexo deploy)

问题

执行hexo d时可能会出现以下错误

fatal: detected dubious ownership in repository at ‘D:/javaa/projects/myhexo/.deploy_git’
‘D:/javaa/projects/myhexo/.deploy_git’ is owned by:‘S-1-5-32-544’but the current user is:
‘S-1-5-21-2495913028-826248603-125958094-1001’ To add an exception for this directory, call:
git config –global –add safe.directory D:/javaa/projects/myhexo/.deploy_git
FATAL { err: Error: Spawn failed
at ChildProcess.(D:\javaa\projects\myhexo\node_modules.store\node_modules\hexo-util\lib\spawn.js:51:21)
at ChildProcess.emit (node:events:513:28)
at cp.emit (D:\javaa\projects\myhexo\node_modules.store\node_modules\cross-spawn\lib\enoent.js:34:29)
at ChildProcess._handle.onexit (node:internal/child_process:291:12) {
code: 128
}
} Something’s wrong. Maybe you can find the solution here: %s https://hexo.io/docs/troubleshooting.html
解决方法：
进入站点根目录 及myhexo目录下 cd D:\javaa\projects\myhexo
删除git提交内容文件夹 rm -rf .deploy_git/
执行 git config –global core.autocrlf false
最后执行 hexo clean && hexo g && hexo d

修改主题

克隆想要使用的主题至themes/文件夹下
git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
修改_config.yml文件,将文件中theme 改为自己想要用的主题
theme: cactus

hexo clean
hexo g
hexo s
hexo d
