ARCH

链接wifi

  1. 查看可用设备

ip link

  1. 将wifi和密码设置到配置文件
    wpa_passphrase PDCN rdxq18019601ls > internet.conf

  2. 开启设备
    ip link set wlp9s0 up

  3. 后台运行
    wpa_supplicant -c internet.conf -i 设备名称 & 设备连接互联网 &表示后台运行

  4. 动态分配ip地址
    dhcpcd &

ping baidu.com

更新系统

pacman -Syyu

安装man:

pacman -S man

安装 base-devel

pacman -S base-devel

添加用户

man useradd / useradd -h 两种方式查看useradd用法

useradd -m -G wheel liushan 新增用户 liushan 并将用户添加到wheel组中

passwd liushan 设置用户密码

编辑sudo文件

sudo vi 使用的是vi命令,vi 命令上下左右间是ABC,最好用vim
sudo vim

如果没有vi命令 使用 一下命令将编辑命令链接到vi
ln -s /usr/bin/vim /usr/bin/vi
将wheel 行解注释

切换到liushan用户 su liushan 或者exit退出root 用liushan账号登录

安装xorg

sudo pacman -S xorg xorg-server

安装deepin

1)
sudo pacman -S deepin deepin-extra
pacman -Qs lightdm 查看是否已安装lightdm

  1. 编辑登录管理器配置文件
    vi /etc/lightdm/lightdm.conf
    修改greeter-session = lightdm-deepin-greeter

  2. 系统启动自动使用lightdm
    sudo systemctl enable lightdm //开机自启动
    or
    sudo systemctl start lightdm
